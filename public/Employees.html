Description

Doing a FIND/READ operation using MongoJS in the cloud database.

Purpose

I have used MongoJS's find() functionality to get data from the backend MongoDB database in my Openshift application and display it in a table.

The purpose of this experiment was to check out how the Mongo operation works in conjunction with NodeJS routes and how to deliver Static HTML pages in the cloud using the express.static command usage to mark static pages for rendering.

You just need to hit the page URL and click on Get Details button to get the data from the backend database in Openshift.
